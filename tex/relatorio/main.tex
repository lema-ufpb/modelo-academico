\begin{resumo}
\noindent Este relatório técnico apresenta os resultados da aplicação de técnicas de \textit{Machine Learning} para a previsão de \textit{churn} de clientes no setor de telecomunicações. Foram comparados três modelos preditivos --- Regressão Logística, \textit{Random Forest} e \textit{LightGBM} --- utilizando um pipeline reprodutível de Ciência de Dados. Os resultados demonstram que o modelo \textit{LightGBM} apresentou desempenho superior, alcançando AUC-ROC de 0,877 e acurácia de 82,5\%. O repositório do projeto contém todos os scripts e dados necessários para a reprodução integral dos experimentos.
\newline
\noindent \textbf{Palavras-chave:} Ciência de Dados. \textit{Machine Learning}. Previsão de \textit{Churn}. Relatório Técnico. Reprodutibilidade.
\end{resumo}

\begin{otherlanguage*}{english}
\begin{resumo}[Abstract]
\noindent This technical report presents the results of applying Machine Learning techniques for customer churn prediction in the telecommunications sector. Three predictive models were compared --- Logistic Regression, Random Forest, and LightGBM --- using a reproducible Data Science pipeline. The results demonstrate that the LightGBM model achieved superior performance, reaching an AUC-ROC of 0.877 and accuracy of 82.5\%. The project repository contains all scripts and data necessary for full reproduction of the experiments.
\newline
\noindent \textbf{Keywords:} Data Science. Machine Learning. Churn Prediction. Technical Report. Reproducibility.
\end{resumo}

A retenção de clientes é um dos principais desafios enfrentados por empresas de telecomunicações no cenário competitivo atual. O fenômeno do \textit{churn}, ou evasão de clientes, representa uma perda significativa de receita, sendo que o custo de aquisição de novos clientes pode ser até sete vezes maior do que o custo de retenção \cite{provost2013}. Neste contexto, a Ciência de Dados oferece ferramentas para a identificação precoce de clientes com alto risco de \textit{churn}.

Este relatório técnico documenta a aplicação de um pipeline de Ciência de Dados para a previsão de \textit{churn}, descrevendo desde a coleta e análise exploratória dos dados até o treinamento e avaliação de modelos preditivos.

\subsection{Modelos de Classificação Supervisionada}

Métodos de classificação supervisionada buscam encontrar uma função que mapeie variáveis preditoras à variável-alvo, minimizando uma função de perda definida \cite{hastie2009}. Dado um conjunto de treinamento $\{(\mathbf{x}_i, y_i)\}_{i=1}^n$, o objetivo é aprender uma função $f: \mathbb{R}^p \to \{0,1\}$ que generalize bem para novos dados.

\subsection{Regressão Logística}

A Regressão Logística modela a probabilidade de pertencimento a uma classe por meio da função sigmoide:

\begin{equation}
    P(Y = 1 \mid \mathbf{X}) = \frac{1}{1 + e^{-(\beta_0 + \boldsymbol{\beta}^T \mathbf{X})}}
    \label{eq:logistic}
\end{equation}

\subsection{Random Forest}

O algoritmo \textit{Random Forest} constrói um conjunto de árvores de decisão a partir de amostras \textit{bootstrap}, selecionando um subconjunto aleatório de preditoras em cada divisão \cite{breiman2001}. A predição final é obtida por votação majoritária.

\subsection{LightGBM}

O \textit{LightGBM}~\cite{ke2017lightgbm} é um método de \textit{gradient boosting} que introduz otimizações como amostragem baseada em gradiente (\textit{GOSS}) e agrupamento exclusivo de \textit{features} (\textit{EFB}), resultando em treinamento mais rápido sem comprometer a acurácia.

\subsection{Métricas de Avaliação}

Para problemas de classificação binária com desbalanceamento de classes, a acurácia isolada pode ser enganosa \cite{geron2022}. As principais métricas utilizadas são AUC-ROC, F1-Score e acurácia balanceada.

\subsection{Pré-processamento}

O pré-processamento incluiu imputação de valores ausentes pela mediana, winsorização de \textit{outliers} nos percentis $P_1$ e $P_{99}$, codificação \textit{one-hot} para variáveis categóricas e normalização com \textit{StandardScaler}.

\subsection{Treinamento e Validação}

Foram treinados três modelos: Regressão Logística ($C = 1.0$), \textit{Random Forest} (200 árvores, $max\_depth = 10$) e \textit{LightGBM} (200 iterações, $learning\_rate = 0.05$). A validação foi realizada por \textit{Stratified K-Fold} com $K = 5$, com divisão treino/teste de 80\%/20\%.
